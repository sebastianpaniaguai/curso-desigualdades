Manejo de la plataforma

Subir archivos

Foto 1. Carga de archivos a la plataforma (2019). Fotografía recuperada de Pixabay

Existen diferentes circunstancias en las que deberás cargar archivos en la plataforma, por ejemplo, cuando debes modificar tu imagen de usuario, subir un documento para una actividad o agregar material de apoyo en una carpeta de documentos compartidos.

La carga de estos archivos debe hacerse a través de una herramienta denominada Selector de Archivos, cuyo procedimiento te explicaremos a continuación.

Foto 2. ¿Cómo cargar archivos en la plataforma? (2019). Fotografía recuperada de Pixabay

El Selector de Archivos te permite elegir archivos y cargarlos en la plataforma. Usualmente, el archivo es tomado del lugar donde lo tienes guardado, como un repositorio o tu computador; sin embargo, existen ciertos escenarios, como el repositorio de videos de YouTube, que crean un enlace y, en algunas circunstancias, un alias o atajo.

Siempre que desees cargar un archivo, aparecerá una herramienta con la siguiente apariencia:

Allí se tienen dos opciones para subir el archivo:

2.1. Cargar un archivo arrastrándolo al área de carga

Animación 1. Cargar un archivo arrastrándolo al área de carga de la plataforma (2019)

2.2. Cargar un archivo desplegando la ventana del Selector de Archivos

Busca la opción Agregar, ubicada en la parte superior izquierda de la herramienta para subir archivos. Aparecerá una ventana emergente denominada Selector de Archivos, que tiene diferentes opciones para seleccionar y adjuntar: un archivo reciente, un archivo nuevo o un archivo privado.

Animación 2. Opción Agregar del selector de archivos (2019)

2.2.1. Archivos recientes

Esta opción te permite elegir entre los últimos archivos que hayas utilizado o cargado en la plataforma.

Para lograrlo, será necesario hacer clic en el archivo que vas a cargar, luego procede a actualizar los campos Guardar como, si deseas agregar otro nombre al archivo; Autor, para identificar el usuario que subió el archivo y Seleccionar licencia para definir los criterios de subida del archivo. Finalmente, bastará con hacer clic en el botón Seleccionar este archivo.

Pasa el mouse por cada ítem para conocer su descripción.

2.2.2. Subir un archivo

Con esta opción es posible cargar un archivo desde el computador o desde una unidad de almacenamiento externa.

Debes hacer clic en el botón Seleccionar archivo, se abrirá una ventana emergente que te permitirá escoger el archivo desde el computador o la unidad de almacenamiento y, para terminar el proceso, haz clic en el botón Abrir.

A continuación se abrirá una ventana para finalizar el proceso. En ella, tienes la posibilidad de validar que el archivo cargó exitosamente, porque su nombre aparece al lado del botón Seleccionar archivo. En este punto, también existe la posibilidad de modificar el archivo, adjuntando uno nuevo. Una vez se confirme haber cargado el archivo correcto, procedes a actualizar los campos; Guardar como, si deseas agregar otro nombre al archivo; Autor, para identificar el usuario que subió el archivo y Seleccionar licencia, para definir los criterios de publicación del archivo. 2.2.3. Archivos privados

Esta opción te permite escoger algún archivo que se encuentre en la carpeta Mis archivos privados. Este es un espacio que la plataforma te proporciona, con el fin de que puedas almacenar archivos, sin necesidad de recurrir a un servicio externo de alojamiento en la nube. Los archivos que subas a este espacio solo estarán disponibles para ti.

Para utilizar esta opción debes hacer clic en el archivo que vas a cargar, luego procedes a actualizar los campos; Guardar como, si quieres agregar otro nombre al archivo; Autor, para identificar el usuario que subió el archivo y Seleccionar licencia, para definir los criterios de publicación del archivo. Para terminar, haz clic en el botón Seleccionar este archivo.
